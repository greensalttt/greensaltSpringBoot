## Commit Message Convention

```
feat: 새로운 기능 추가
fix: 버그 수정
test: 테스트 코드
remove: 파일 삭제
docs: README 변경
style: 코드 스타일 변경
chore: 빌드 업데이트 및 자잘한 수정
refactor: 코드 구조 개선 (기능 변경 X)
```
